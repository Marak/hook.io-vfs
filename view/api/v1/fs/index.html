Multi-Cloud File Storage

A simple File System API capable of communicating with all major cloud file storage providers

Hook.io's File API allows developers to seamlessly work with files from multiple remote storage providers

  • Amazon S3
  • Google Cloud
  • Rackspace Cloudfiles
  • Microsoft Azure
  • Any SSH / SFTP server
